# Project Structure

```
├── assets
│   ├── fonts
│   └── sprites
├── data
├── docs
├── README.md
├── scenes
└── scripts
```

## Assets

Contains all the assets for the game. This includes sprites, tiles, sounds, music, and other assets.

## Data

Contains all the game state, and game configurations.

## Docs

Documentation directory.

## Scenes

Contains all the scenes for the game. This includes the main menu, the game world, the player, the buildings, the animals, the crops, and other scenes.

## Scripts

Contains all the scripts related to scenes and other logic.
